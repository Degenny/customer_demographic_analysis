Customer Demographics Analysis

Project Overview
This project analyses customer demographic data to understand the relationship between family size and educational qualification.

The goal is to identify whether certain family size groups are associated with higher or lower education levels and to ensure the analysis is not biased by uneven sample distribution.
Business Problem
How does family size relate to educational attainment, and are initial conclusions affected by dataset distribution bias?
Dataset
The dataset includes:
- Family size group (Small, Medium, Large)
- Educational qualification
- Gender
- Income category
Tools Used
- SQL (data aggregation and normalization)
- Excel (data cleaning and pivot tables)
- Power BI (data visualization)
Methodology
1. Initial analysis using raw counts
2. Identification of sample size imbalance across groups
3. Normalization using percentage distribution within each group
4. Comparison between raw and normalized results
Key Insight
Initial results suggested that medium-sized families dominate educational outcomes. However, after normalization, it became clear that this 
was driven by sample size imbalance rather than true distribution differences.
Conclusion
This project demonstrates how relying on raw counts can lead to misleading conclusions, and why normalization is essential in demographic analysis.
